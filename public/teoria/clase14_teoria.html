3 Del Clip a la Secuencia Narrativa

Los 3 niveles del contenido de vídeo

Nivel Qué es Ejemplo Complejidad
Clip Fragmento de 3-10 seg Rotación de producto Básico
Secuencia 3-5 clips conectados con narrativa "Del boceto al zapato" Intermedio
Pieza final Vídeo completo con intro, desarrollo, cierre, audio Spot de campaña de 30 seg Avanzado

Estructura narrativa para vídeos de producto de lujo

CODE 
ACTO 1: MISTERIO (0-5 seg)
Silueta oscura, luz que se acerca, textura en macro → INTRIGA

ACTO 2: REVELACIÓN (5-15 seg)  
El producto emerge completamente, rotación, detalles → DESEO

ACTO 3: CONTEXTO (15-25 seg)
El producto en escena (lifestyle, modelo, tienda) → ASPIRACIÓN

ACTO 4: CIERRE (25-30 seg)
Logo, CTA, silencio elegante → ACCIÓN

🎬 **Regla del lujo:** Menos es más. El lujo se comunica con ritmo lento, silencios, movimiento preciso. Un vídeo de marca premium NUNCA debería parecer un anuncio de Black Friday.

4 Storyboarding con IA

Cómo planificar antes de generar

Antes de generar clips, planifica la secuencia completa usando IA como co-director:

Prompt de storyboarding (para Gemini/Claude)

CODE 
Actúa como director creativo de un spot publicitario para una marca de 
zapatos de lujo. Crea un storyboard de 6 escenas para un vídeo de 30 
segundos con la siguiente narrativa:

"Del artesano a tu armario — El viaje de un zapato hecho a mano"

Para cada escena incluye:
1. Número de escena y duración (en segundos)
2. Descripción visual detallada
3. Movimiento de cámara
4. Mood/Atmósfera  
5. Audio/música sugerido
6. Prompt de generación I2V listo para usar

Tono: Hermès meets Apple. Minimalista pero emotivo.

5 Transiciones y Morphing

Conectar clips con elegancia

Las transiciones entre clips son lo que diferencia un vídeo amateur de uno profesional.

Tipos de transición con IA

Tipo Cómo funciona Herramienta ideal Efecto
Morphing Un objeto se transforma en otro Luma Dream Machine Oxford → Stiletto fluidamente
Match Cut Misma composición, diferente contenido Runway (2 clips I2V similares) Zapato en mesa → zapato en pasarela
Fade Fundido a negro/blanco entre clips Cualquier editor Elegante, clásico
Slide Un clip empuja al siguiente Editor de vídeo Dinámico, moderno

Morphing entre productos con Luma

CODE 
Keyframe A: [Render Oxford marrón en mármol]
Keyframe B: [Render Stiletto negro en mármol]
Prompt: "Ultra-smooth morphing transition, luxury product transformation, 
same camera angle, same lighting, the shoe elegantly metamorphoses"

Ejemplo de prompt para música de campaña

CODE 
Create a 30-second ambient music track for a luxury shoe brand commercial.
Style: minimal piano + soft strings + subtle electronic elements.
Mood: elegant, sophisticated, modern, aspirational.
Tempo: slow (70 BPM). No vocals. Cinematic quality.
Build: starts whisper-quiet, crescendo at 15 seconds, softens for logo reveal.

7 Lookbook Animado

La Pieza Final: Tu Catálogo Cobra Vida

Un lookbook animado es la evolución del catálogo de producto estático. Combina:

  • Renders de producto animados (rotación, movimiento)
  • Transiciones elegantes entre modelos
  • Tipografía y datos de producto superpuestos
  • Música y voz de marca

Estructura del Lookbook Animado

CODE 
00:00 — PORTADA ANIMADA
        Logo + nombre de colección con movimiento sutil

00:05 — MODELO 1 (OXFORD)
        Rotación 360° + ficha de producto superpuesta
        Transición: morphing a modelo 2

00:15 — MODELO 2 (STILETTO)
        Hero shot con Motion Brush (fondo vivo)
        Transición: match cut

00:25 — MODELO 3 (MOCASÍN)
        Macro detalle → zoom out a composición completa
        Transición: fade elegante

00:35 — MODELO 4 (ANKLE BOOT)
        Lifestyle: modelo caminando
        Transición: slide

00:45 — MODELO 5 (SNEAKER)
        Secuencia rápida de 3 colores
        Transición: morphing a cierre

00:55 — CIERRE
        Logo + CTA + URL + música fadeout

8 Post-producción

Herramientas IA para el Acabado Final

Tarea Herramienta IA Qué hace

9 Quiz y PFC

📝 Quiz de Conocimientos (8 preguntas)

**1.** ¿Cuáles son los 4 actos de la estructura narrativa para vídeos de producto de lujo?

  • a) Intro, nudo, desenlace, créditos
  • b) ✅ Misterio (intriga), Revelación (deseo), Contexto (aspiración), Cierre (acción)
  • c) Teaser, demo, testimonial, venta
  • d) Fade in, clip, transición, fade out

**2.** ¿Qué herramienta es ideal para crear un morphing entre dos productos?

  • a) Runway Motion Brush
  • b) ✅ Luma Dream Machine (keyframes)
  • c) ElevenLabs
  • d) CapCut solo

**3.** ¿Qué hace ElevenLabs?

  • a) Genera imágenes
  • b) ✅ Convierte texto en voz profesional y permite clonar voces para consistencia de marca
  • c) Edita vídeo
  • d) Genera música

**4.** ¿Qué es un "Shot List"?

  • a) Una lista de fotos de stock
  • b) ✅ Lista planificada de cada plano/escena del vídeo con descripción, cámara, duración y prompt
  • c) Una colección de screenshots
  • d) Los clips descartados

**5.** ¿Cuánto debería durar típicamente un lookbook animado de 5 productos?

  • a) 5 segundos
  • b) ✅ 45-60 segundos (8-12 seg por producto + intro/cierre)
  • c) 5 minutos
  • d) 30 minutos

**6.** ¿Para qué sirve el Frame Interpolation (Runway)?

  • a) Para añadir marcos a las imágenes
  • b) ✅ Para crear slow-motion fluido generando fotogramas intermedios que no existían
  • c) Para comprimir el vídeo
  • d) Para añadir texto

**7.** La regla del lujo en vídeo dice:

  • a) Más efectos = más premium
  • b) Vídeos largos son mejores
  • c) ✅ Menos es más: ritmo lento, silencios, movimiento preciso
  • d) Siempre usar música alta

**8.** ¿Qué herramienta usarías para unificar la paleta de color entre 5 clips generados con diferentes herramientas IA?

  • a) Voy a regenerar todos con la misma herramienta
  • b) ✅ Color Grading en DaVinci Resolve o CapCut para unificar la estética
  • c) Cambio los prompts
  • d) No se puede hacer

🚀 PFC — Preparación para Lab de Vídeo II (Clase 15)

Prepara el material para el lookbook animado y la pieza de campaña.

  1. Completar el storyboard del lookbook animado (6-8 escenas)
  2. Seleccionar los 5 renders que aparecerán en el lookbook
  3. Escribir todos los prompts I2V para cada escena
  4. Elegir la música (buscar en Epidemic Sound o generar con Suno AI)
  5. Instalar CapCut o preparar DaVinci Resolve para post-producción

📦 **Entregable:** Storyboard completo (6-8 escenas) + 5 renders seleccionados + prompts I2V + referencia de música.
